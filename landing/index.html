Why it exists

Unused skills aren't free.

Every skill your agent loads is text it reads past — and pays for — on every turn.

▲

Context rot

Frontier models lose 30%+ accuracy at mid-window positions long before the window is "full." Noise degrades answers.

▲

The discovery ceiling

Past ~32 installed skills, Claude Code truncates even the skill descriptions. Native lazy-loading was closed "not planned."

▲

Libraries keep growing

Superpowers, GSD, marketplaces — skill counts only go up. Every install makes every turn heavier.

What you get

A scalpel, not a switch.

●

Fully local

BM25 ranking + trigger rules. Runs in milliseconds, no GPU, no model download. Nothing leaves your machine.

●

Zero paid calls

Other skill routers bill you Haiku tokens per prompt. slimcontext costs nothing to run — ever.

●

Savings dashboard

slimcontext stats shows cumulative tokens saved. No other skill tool ships this.

●

Reversible

Skills are moved, never deleted. One restore brings the whole library back.

●

In-editor menu

Type /slimcontext in Claude Code — slim, restore, toggle, or check savings without leaving.

●

Quality-safe mode

The advisory hook never removes a skill — it only scores and nudges, so it cannot hurt an answer.

Quick start

Lean in three commands.

1

Install

Node 18+, no native modules.

npm install -g slimcontext
2

Stage your skills for the task ahead

Parks irrelevant skills so the next session starts lean. Reversible anytime.

slimcontext apply "refactor the auth module"
3

Watch the wins add up

Or run slimcontext init and let the hook do it.

slimcontext stats
slimcontext does not touch MCP tool loading — Claude Code shipped native "MCP Tool Search" in January 2026. This tool is about your skills.
